# U.I Khaleel Agrilink Solutions

Official website — connecting farmers, markets and innovation across Nigeria.

## Run locally

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deploy to Vercel

1. Push this repo to GitHub.
2. Go to vercel.com → New Project → Import this repo.
3. Click Deploy. No configuration needed.
4. (Optional) Add `OPENAI_API_KEY` under Project → Settings → Environment Variables to power the AI Assistant with real responses.
